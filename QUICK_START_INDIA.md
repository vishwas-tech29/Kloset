# 🚀 Quick Start - India Version

## One Command Setup

### Windows:
```bash
setup-india.bat
```

### Mac/Linux:
```bash
chmod +x setup-india.sh && ./setup-india.sh
```

### Manual:
```bash
npx tsx prisma/seed-india.ts && npm run dev
```
