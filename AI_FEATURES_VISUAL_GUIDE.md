## 🎯 State Management

```
┌─────────────────────────────────────────┐
│  TryOnStudio Component State            │
├─────────────────────────────────────────┤
│  step: 'upload' | 'select' | 'result'   │
│  userPhoto: string | null               │
│  selectedProduct: Product | null        │
│  resultImage: string | null             │
│  isProcessing: boolean                  │
└─────────────────────────────────────────┘
         │
         ├─→ PhotoUploader
         │   └─→ onPhotoUpload(url)
         │
         ├─→ GarmentSelector
         │   └─→ onProductSelect(product)
         │
         └─→ TryOnResult
             └─→ onReset() / onTryAnother()
```
